Help & Guide

➕ Adding Events

Click + Add Event in the header. Fill in the title (required), an optional description (up to 1000 characters — a live counter shows remaining space), and the date & time. Hit Save Event — it saves to Google Sheets and appears instantly.

⏱ Countdown & Elapsed Timers

Each event card shows a live timer split into months · days · hours · mins · secs. Upcoming events count down; past events count up. Timers refresh every second automatically.

📋 Event Title List (Accordion)

The collapsible Event Title List panel above the cards gives you a compact overview of all filtered events at a glance — with status badges and dates. Click any row to jump straight to editing that event. Click the panel header to expand or collapse it.

📊 Dashboard Stats

Four stat cards show: Total Events, Upcoming (future), Past (elapsed), and Months Span — the number of months between your earliest and latest event, giving a sense of your timeline spread.

🔍 Search & Filter

Use the search bar to filter across title, description, and date. Use the All / Upcoming / Past tabs to narrow by status, and the sort dropdown to order by date or creation time — all update in real-time.

📥 Export Events

Go to Settings → Data → Export Events as CSV to download all your events as a spreadsheet-compatible CSV file.

☁️ Data Storage

Events sync to a Google Sheet via Google Apps Script. Set your Web App URL and API Key under Settings → API Configuration. Without an API, events fall back to browser local storage automatically.

🔐 Password & Security

A password protects the dashboard. On first launch you'll set one. To change it later, use Settings → Security. All passwords are SHA-256 hashed and stored only in your local browser — never sent to any server.
