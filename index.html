Help & Guide

➕ Adding Events

Click + Add Event in the header. Fill in the title (required), an optional description, and the date & time. Hit Save Event — it syncs to Google Sheets and appears instantly.

⏱ Countdown & Elapsed Timers

Each event card shows a live timer split into months · days · hours · mins · secs. Upcoming events count down; past events count up. Timers refresh every second automatically.

📋 Event List (Accordion)

The Event List panel gives a compact overview — status badges and dates. Click any row to expand the full card. Only one row expands at a time.

📊 Dashboard Stats

Four stat cards show: Total Events, Upcoming, Past, and Months Span — the range between your earliest and latest event.

🔍 Search & Filter

Search bar filters across title, description, and date. Tabs narrow by status. Sort dropdown orders results — all update in real-time.

☁️ Data Storage

Events sync to a Google Sheet via Google Apps Script. Set your Web App URL and API Key under Settings → API Configuration. Without an API, events fall back to browser localStorage automatically.

🔐 Security

A password protects the dashboard. Passwords are SHA-256 hashed and stored only in your local browser — never sent to any server. Use Settings → Session → Lock to sign out.
